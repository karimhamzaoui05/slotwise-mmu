# SlotWise MMU

SlotWise MMU is a campus resource booking system for study rooms, lab equipment, and shared project spaces.

The app will focus on:

- Real-time availability for bookable resources
- Conflict-safe booking creation, editing, and cancellation
- Waitlist handling when a time slot is full
- A simple admin view for managing resources and blocked times

This project is being built for the Shortcut Asia Internship Challenge 2026.
